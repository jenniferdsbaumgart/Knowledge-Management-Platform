Knowledge Platform Widget Demo

This page demonstrates the embeddable search widget. Click the search button in the bottom-right corner to try it out.

Installation

<script src="https://your-domain.com/widget.js"></script>
<knowledge-search
  api-url="https://api.example.com"
  api-key="your-api-key"
  theme="light"
  position="bottom-right"
></knowledge-search>

Attributes

  • api-url - Base URL for the Knowledge Platform API
  • api-key - Optional API key for authenticated requests
  • theme - "light" or "dark"
  • position - "bottom-right" or "bottom-left"
